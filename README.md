# MOSSAE 밤길안심서비스

## 앱 간단 설명
>밤길안전과 범죄 예방을 위한 밤길안심서비스로 범죄의 대상이 되는 것을 예방하고,
>비상전화, 사이렌, 페이크콜 기능 등으로 위험할 때 도움을 줄 수 있는 앱을 개발하였다.


## 수상
>2018 교내 학과 경진대회 우수상


## 기간
>2018.04~2018.06

## 개발 설명

개발인원 : 2명

개발언어 : Java
